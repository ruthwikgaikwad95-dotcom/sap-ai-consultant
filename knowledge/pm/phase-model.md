# Phase-Based Maintenance (PBM)

Module: SAP PM

Availability:

* SAP S/4HANA Asset Management
* Not available in classic ECC PM

Important:

* Phase-Based Maintenance (PBM) is different from classical maintenance planning.
* Do not use transaction IP10 to configure PBM.
* IP10 is used to schedule maintenance plans.
* PBM is part of SAP S/4HANA Asset Management innovations.

Business Function:

* Verify business function activation in your S/4HANA release before implementation.

Answering Rules:

* If a user asks about Phase-Based Maintenance, clearly state that IP10 is not used to configure PBM.
* Do not invent SAP tables.
* Do not invent SPRO paths.
* If uncertain, explicitly state that verification in SAP Help Portal is required.
